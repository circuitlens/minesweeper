# Minesweeper
Minesweeper coded in Python for a school assignment. Python must be installed on your computer to play, controls same as the original game (left-click to reveal, right-click to mark/unmark a mine with a flag). The player can adjust the difficulty of the game by changing the amount of rows and columns the game's grid has, as well as adjusting the amount of mines generated. The game automatically finishes once all mines are correctly flagged and all unmined squares are revealed, or when the player blows up. The game tracks the amount of wins and losses, which the player can save and load if they so desire. Unlike the original game, there is so far no prevention for a mine being on the very first square the player clicks - this missing feature may be addressed later.
